■操作方法
カーソルの変わらないところをダブルクリック・・・メニュー
カーソルの変わらないところをホイールクリック・・・ランダムトーク
肩をダブルクリック・・・話しかけるメニュー
それ以外はさわり判定です。

■好感度などの確認方法
このゴーストはSSPのAIグラフに対応しています。
好感度などの確認には右クリックメニューの「情報」から「好感度などの確認」をクリックします。

■おまけ
cキーを押すとパラメータを変更することが出来る画面が開きます。
キャラクターを素直に楽しみたいひとは使わないことを強く推奨します。
